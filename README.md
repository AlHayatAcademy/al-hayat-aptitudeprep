# Al-Hayat Vocabulary Batch 10

This release adds 100 reviewed learning records and 173 original MCQs while preserving the base question bank and Batches 02-09.

## Upload through GitHub Desktop

1. Extract this ZIP.
2. Copy the extracted `assets` and `data` folders into the main `al-hayat-aptitudeprep` repository folder.
3. Choose **Replace files** when prompted.
4. Commit with: `Add reviewed vocabulary batch 10`
5. Push, wait for GitHub Pages deployment, and hard-refresh the website.

## Expected live result

- 954 ready-to-learn vocabulary records
- Approximately 1,813 vocabulary MCQs after all indexed files merge on the current live site
- 182 ten-question sets; the final set contains 3 questions
- 3,000 entries remain in the master curriculum

## Content quality

- 100 unique, meaningful exemplar sentences
- Exemplar sentence retained in every answer explanation
- 10 textbook-informed examples with exact source IDs
- All remaining examples are original aptitude-preparation sentences
- Textbook-informed sentences are original, sense-matched adaptations, not copied text
- Complete meanings, synonyms, defensible antonyms, word families, collocations, usage notes, Urdu support, and IPA
- Exact 10% Foundation, 35% Intermediate, 40% Advanced, and 15% Challenge mix
- 173 MCQs: 50 direct synonyms, 60 meanings in context, 40 antonyms, and 23 word-family questions
- Invalid curriculum artifact `bia` quarantined because it duplicates the already released headword `bias`
