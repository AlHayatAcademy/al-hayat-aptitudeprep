# Al-Hayat Vocabulary Batch 12

This release adds 100 reviewed learning records and 173 original MCQs while preserving the base question bank and Batches 02-11.

## Upload through GitHub Desktop

1. Extract this ZIP.
2. Copy the extracted `assets` and `data` folders into the main `al-hayat-aptitudeprep` repository folder.
3. Choose **Replace files** when prompted.
4. Commit with: `Add reviewed vocabulary batch 12`
5. Push, wait for GitHub Pages deployment, and hard-refresh the website.

## Expected live result

- 1,154 ready-to-learn vocabulary records
- Approximately 2,159 vocabulary MCQs after all indexed files merge on the current live site
- 216 ten-question sets; the final set contains 9 questions
- 3,000 entries remain in the master curriculum

## Content quality

- 100 unique exemplar sentences; every answer explanation retains its example
- 16 traceable textbook-informed examples; all are original sense-matched adaptations
- Complete meanings, synonyms, antonyms, word families, collocations, usage notes, Urdu support, and IPA
- Exact 10% Foundation, 35% Intermediate, 40% Advanced, and 15% Challenge mix
- 173 MCQs: 50 direct synonyms, 60 contextual meanings, 40 antonyms, and 23 word-family questions
- Invalid curriculum artifact `bia` remains quarantined and unreleased

## Preserve existing files

Preserve `data/questions.json` and every `data/questions-vocabulary-batch-02.json` through `data/questions-vocabulary-batch-11.json`. The updated index adds Batch 12 without replacing earlier question files.

## Review status

English meanings, examples, tested senses, options, and answers were editorially checked. Urdu meanings and pronunciation approval remain review-gated and are not represented as human-approved.
